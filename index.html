Vogue

Auto-reload stylesheets whenever CSS files are saved.

by Andrew Davey

Watch Vogue in action

Two minute screencast showing life without and then with Vogue.

Install

You will need to have NodeJS and npm installed.

Install with a simple:

npm install vogue

Yep, that's it.

Usage

Start Vogue by running a command like this:

vogue path/to/website

This runs an HTTP server on port 8001.

You can also specify the port:

vogue --port 1234 path/to/website
